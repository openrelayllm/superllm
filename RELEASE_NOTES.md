# Release Notes

## v0.39.0 - 2026-07-08

### 新增

- 新增后台核心配置导入/导出能力，支持导出迁移 JSON、导入前预览、upsert 导入和迁移范围说明。
- 新增前端“导入/导出”页面，支持文件选择、拖拽/粘贴 JSON、预览核心表、确认导入和导入结果展示。
- 供应商新增本地账号绑定、API 端点候选、能力状态、平台识别、URL 提示、操作提示和集成提示信息。
- NewAPI 适配新增账号、分组、Key 和会话探测增强，为供应商运营页提供更完整的可操作状态。

### 改进

- 拆分供应商 service 逻辑，将账号、凭据、过滤、能力、归一化和提示规则收敛到独立文件，降低单文件复杂度。
- 扩展采集增强 NewAPI、Sub2API 和站点探测数据，回传更完整的供应商地址、登录、API 和候选信息。
- 运营供应商表格补充最佳通道动作和状态呈现，便于直接判断供应商接入质量。
- 调度、成本、会话和站点发现流程补充供应商能力字段，统一后端和前端展示口径。

### 修复

- 修正供应商筛选、创建、更新和站点匹配在多平台提示下的边界处理。
- 修正 NewAPI 会话、分组和 Key 同步中的兼容性问题，减少不同 NewAPI 站点返回结构差异导致的采集失败。

### 测试

- 增加导入/导出 service 与 handler 测试。
- 增加供应商账号、凭据、创建、更新、过滤、站点候选和 domain 归一化测试。
- 增加 NewAPI provider、供应商路由、运营页和侧边栏路由回归测试。

### 发布

- 更新版本号到 `0.39.0`。
- GitHub Release 继续只发布 Linux 产物：`linux_amd64`、`linux_arm64` 和 `checksums.txt`。
- DockerHub/Railway 镜像渠道不随常规发布自动执行；如需要，单独走镜像渠道发布流程。
