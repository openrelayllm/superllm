# Release Notes

## v0.15.0 - 2026-06-26

### 新增

- 新增代理管理模块，支持代理订阅、节点、策略、目标规则、运行槽、分配记录和审计日志。
- 新增站点发现代理策略选择，采集任务可按策略绑定代理出口并在任务结束后释放。
- 新增 new-api / sub2api 后端直注册能力，支持验证码读取、注册结果入库和浏览器兜底。
- 新增注册流程列表与注册日志，可查看注册状态、重跑失败注册并追踪业务日志。
- 新增系统日志页面，集中查看余额、注册、插件和邮件验证码等 Admin Plus 业务日志。

### 改进

- 站点发现注册流程支持并发保护、状态复用、失败重试和注册完成后自动创建供应商。
- 余额采集补充业务日志与 HTTP 诊断，便于定位供应商接口、会话和上游响应问题。
- 供应商会话探测和注册 provider 增加代理 URL 支持，为跨区域注册和探测提供统一出口。
- 采集网址、注册列表和注册流程的前端状态展示拆分为 presentation helper，减少重复判断。

### 修复

- 修复注册流程重复触发时可能创建多条任务的问题。
- 修复部分注册失败场景缺少结构化错误码和日志上下文的问题。
- 修复已注册候选的导入按钮、处理状态和注册状态展示不一致的问题。

### 发布

- 更新版本号到 `0.15.0`。
- GitHub Release 继续只发布 Linux 产物：`linux_amd64`、`linux_arm64` 和 `checksums.txt`。
- DockerHub 镜像继续由 GitHub Actions 发布，不依赖本地 Docker。
