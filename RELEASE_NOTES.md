# Release Notes

## v0.14.0 - 2026-06-26

### 新增

- 新增 Gmail OAuth 凭据管理与验证码读取页面，支持授权、凭据检查、测试发送和轮询读取验证码。
- 新增邮箱验证码后台能力，覆盖 Gmail readonly 访问、邮件模板分类、验证码提取和错误码诊断。
- 新增站点发现注册任务的延迟建档流程，允许插件先完成供应商注册，再由后台落库绑定供应商。
- 新增插件上报供应商候选接口，可从当前站点采集页面信息、登录凭据和充值入口并沉淀为供应商。

### 改进

- 注册任务接入邮箱验证码读取，插件可通过任务租约获取验证码并回传注册结果。
- 供应商注册凭据支持注册完成后再绑定供应商，减少未完成注册对目录数据的污染。
- 插件供应商采集和会话采集职责拆分，显式要求会话任务传入供应商 ID，避免隐式创建带来的歧义。
- 后台侧边栏新增“邮箱验证”入口，统一管理 Gmail 授权和验证码读取能力。

### 修复

- 修复站点发现注册任务必须预先存在供应商 ID 的限制，支持真正的 deferred registration。
- 修复插件注册失败结果无法回写注册凭据状态的问题。

### 发布

- 更新版本号到 `0.14.0`。
- GitHub Release 继续只发布 Linux 产物：`linux_amd64`、`linux_arm64` 和 `checksums.txt`。
- DockerHub 镜像继续由 GitHub Actions 发布，不依赖本地 Docker。
