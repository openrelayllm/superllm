# Release Notes

## v0.9.2 - 2026-06-22

### 修复

- 成本对账快照按实际余额推导用量消耗和余额差异，避免上游 usage 原始消耗与余额快照不一致时出现错误差异。
- 成本台账 upsert 从冲突忽略改为冲突更新，已同步的权益类型、金额和原始载荷修正可以回写。
- 兑换记录区分余额、并发和订阅权益；并发和订阅权益不再计入余额充值金额。
- 本地 Sub2API 开通链路收敛为只确保账号，不再创建或绑定本地 group，减少与本地分组模型的耦合。

### 更新

- 成本页将充值总额拆为充值订单和兑换充值，并统一展示用量消耗、实际余额和余额差异。
- 本地账号默认启用 `pool_mode=true`，账号补齐逻辑会保留既有 group 绑定。
- 账号路线图文档从 group/account 并存口径收敛为 account 口径。

### 发布

- 更新版本号到 `0.9.2`。
- GitHub Release 继续只发布 Linux 产物：`linux_amd64`、`linux_arm64` 和 `checksums.txt`。
- DockerHub 镜像继续由 GitHub Actions 发布，不依赖本地 Docker。
