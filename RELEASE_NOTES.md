# Release Notes

## v0.31.0 - 2026-07-01

### 新增

- 站点目录支持后台物理删除，删除时同步清理目录关联并将来源发现记录回退为待处理。
- 供应商成本支持一键历史回补，通过调度器串行生成每个供应商的成本同步任务。

### 改进

- 站点目录后台列表不再被 1000 条读取上限截断，分页 total 会反映完整目录数量。
- 供应商成本页面拆分总览、供应商快照和单供应商明细，减少首屏明细拉取压力。
- 调度步骤保存请求和结果快照，方便追踪成本回补任务执行结果。

### 测试

- 增加站点目录物理删除、路由暴露和 SQL 事务行为回归测试。
- 增加供应商成本历史回补调度与步骤结果快照回归测试。

### 发布

- 更新版本号到 `0.31.0`。
- GitHub Release 继续只发布 Linux 产物：`linux_amd64`、`linux_arm64` 和 `checksums.txt`。
